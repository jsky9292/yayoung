야구 글러브 가격 비교 및 크롤링 시스템

📖 프로젝트 소개

야용사 글러브 마켓은 다양한 소스(야후옥션, 다음 카페)에서 야구 글러브 정보를 수집하고 가격을 비교할 수 있는 웹 기반 크롤링 시스템입니다.

🌐 다중 소스 크롤링

야후옥션, 야용사 카페 등 여러 플랫폼에서 실시간 데이터 수집

💰 가격 비교

환율 자동 계산 및 배송비, 관세 포함 총 비용 산출

📊 대시보드

수집된 데이터를 시각화하여 한눈에 확인 가능

🔍 필터링

브랜드, 포지션, 가격대별 상세 검색 기능

🛠️ 기술 스택

Python 3.x Flask Selenium BeautifulSoup SQLite JavaScript HTML/CSS Chart.js

🚀 주요 기능

  • 야후옥션 크롤링: 일본 야후옥션에서 야구 글러브 정보 수집
  • 야용사 카페 크롤링: 다음 카페에서 중고/신품 글러브 정보 수집
  • 자동 가격 계산: 엔화 → 원화 환율 적용, 배송비 및 관세 자동 계산
  • 브랜드 인식: Mizuno, Rawlings, SSK, ZETT 등 주요 브랜드 자동 분류
  • 이미지 수집: 상품 이미지 자동 다운로드 및 저장
  • 데이터 분석: 수집된 데이터 기반 통계 및 시각화

📦 설치 및 실행

# 저장소 클론
git clone https://github.com/jsky9292/yayoung.git
cd yayoung

# 가상환경 생성 및 활성화
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 의존성 설치
pip install -r requirements.txt

# Flask 앱 실행
python main_app.py

# 브라우저에서 접속
# http://localhost:5000
